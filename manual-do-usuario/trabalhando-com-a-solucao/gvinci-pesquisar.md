# Gvinci : Pesquisar

A combinação **&lt;Ctrl&gt; + &lt;F&gt;** é utilizada para abrir a caixa "Pesquisar", que possibilita a procura de itens na solução.

A caixa de diálogo abaixo é exibida.

![](http://www.gvinci.com.br/manual/pesquisa.zoom80.png)

**Texto para pesquisa:** Insira o texto que será procurado.

**Pesquisar em:** Permite definir o local exato para a pesquisa:

![](http://www.gvinci.com.br/manual/pesquisa2.zoom80.png)

**Diferenciar maiúsculas e minúsculas:** É utilizada para que o item procurado esteja no mesmo formato \(maiúsculas ou minúsculas\) que o item encontrado.

**Procurar texto completo:** É usada para procura da palavra completa do item procurado.

Após as definições da pesquisa, clique em ![](http://www.gvinci.com.br/manual/pesquisarbt412.png). O botão ![](http://www.gvinci.com.br/manual/resultadosbusca412.png) aparecerá abaixo do painel Módulos da solução exibindo a quantidade de itens encontrados. Clique neste botão e a lista dos itens encontrados será exibida:

![](http://www.gvinci.com.br/manual/procurargv5.zoom80.png)

Para exibir a localização do item, clique duas vezes sobre a linha desejada.
